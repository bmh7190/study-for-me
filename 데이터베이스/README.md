# 데이터베이스

관계형 모델, SQL, ER 모델, 정규화, 저장 구조, 인덱싱, 쿼리 처리를 정리한 노트입니다.

## 목차

- [Intro](<Intro.md>)
  - 데이터베이스와 DBMS의 기본 개념, SQL과 Vector DB 같은 입문 주제를 간단히 다룹니다.
- [1. Relational Model](<1. Relational Model.md>)
  - relation, schema, key, integrity constraint 등 관계형 모델의 기본 구성 요소를 정리합니다.
- [2. SQL Basics](<2. SQL Basics.md>)
  - SELECT, INSERT, UPDATE, DELETE를 중심으로 기본 SQL 문법과 질의 작성 방식을 다룹니다.
- [3. Intermediate SQL](<3. Intermediate SQL.md>)
  - join, nested query, aggregation, view 등 조금 더 복합적인 SQL 사용법을 정리합니다.
- [4. Entity-Relationship Model](<4. Entity-Relationship Model.md>)
  - entity, relationship, attribute, cardinality를 통해 데이터베이스 설계의 개념적 모델링을 다룹니다.
- [5. Relational DB Design](<5. Relational DB Design.md>)
  - functional dependency, lossless decomposition, BCNF, 3NF 등 관계형 스키마 정규화를 정리합니다.
- [6. Storage and File Structure](<6. Storage and File Structure.md>)
  - storage hierarchy, file organization, record 구조, buffer manager와 replacement policy를 다룹니다.
- [7. Indexing](<7. Indexing.md>)
  - index의 필요성, ordered index, B+ tree, hashing 등 검색 성능을 높이는 자료구조를 정리합니다.
- [8. Query Processing](<8. Query Processing.md>)
  - query evaluation, selection, join 처리, cost estimation 등 SQL이 실행되는 내부 과정을 다룹니다.
