
## **Recap: Basic Query Structure**


일반적인 SQL의 쿼리는 다음과 같은 형태를 가진다. 

```sql
select A1, A2, ...,An from r1, r2, ..., rm where P
```

```sql
select name
from instructor
where dept_name = 'Comp. Sci.' and salary > 80000
```

• A<sub>i</sub> : 속성 
• R<sub>i</sub> : 릴레이션 
• P : 조건

SQL 쿼리의 결과는 릴레이션이다.
