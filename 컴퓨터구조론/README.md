# 컴퓨터구조론

컴퓨터 시스템의 추상화, 명령어 집합, 산술 연산, 프로세서, 메모리 구조를 정리한 노트입니다.

## 목차

- [1. Computer Abstractions and Technology](<1. Computer Abstractions and Technology.md>)
  - 컴퓨터 시스템의 계층적 추상화, 성능 지표, 응답 시간과 처리량의 차이를 다룹니다.
- [2. Instructions_Language of the Computer](<2. Instructions_Language of the Computer.md>)
  - MIPS 명령어 집합을 중심으로 산술, 메모리 접근, 분기, 프로시저 호출, 주소 지정 방식을 정리합니다.
- [3. Arithmetic for Computers](<3. Arithmetic for Computers.md>)
  - 정수와 실수 표현, 산술 연산, overflow, floating-point 연산의 기본 원리를 다룹니다.
- [4. Processor](<4. Processor.md>)
  - datapath와 control을 중심으로 명령어가 프로세서 내부에서 실행되는 흐름을 정리합니다.
- [5. Memory](<5. Memory.md>)
  - 메모리 계층 구조, cache, 가상 메모리 등 저장 장치가 성능에 미치는 영향을 다룹니다.
