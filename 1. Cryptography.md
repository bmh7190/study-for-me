**Cryptography란 악의적인 공격자에 대해 안전한 통신을 가능하게 하기 위한 기술이다.**

Cryptography에서 **cryptographic primitives**는 컴퓨터 보안 시스템에서 사용되는 기본적인 저수준 알고리즘으로, 다양한 보안 기능을 구현하기 위한 핵심 구성 요소이다. 이러한 primitive들은 기밀성, 무결성, 인증과 같은 개별적인 기능을 수행하며, 독립적으로 사용되기보다는 상위 구조를 구성하는 데 활용된다.

한편, **cryptographic protocol**은 이러한 primitive들을 조합하여 실제 보안 목적을 수행하도록 설계된 절차 또는 규칙이다. 즉, 여러 암호학적 기법을 일정한 순서로 적용하여 통신 보안, 인증, 키 교환과 같은 보안 기능을 구현한다.

결과적으로, cryptographic primitives가 개별적인 기능을 수행하는 기본 도구라면, cryptographic protocol은 이들을 조합하여 실제 보안 시스템을 구성하는 상위 개념이라고 할 수 있다.
