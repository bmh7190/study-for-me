## HTTP
hyper transfer protocol 의 약자로 Web 의 어플리케이션 계층의 프로토콜이다. 클라이언트와 서버 모델로 나눌 수 있느넫, 일반적으로 client는 HTTP를 사용해서 서버에게 요청을 보내고, 응답을 받아서 Web 객체들을 화면에 띄운다. 서버는 요청에 대한 응답을 보내는 모델로, 여기서도 http 로 보낸다..

HTTP는 TCP를 사용한다. 클라이언트가 TCP 연결을 요청하면 서버는 요청을 받는 3handshake한다. 그 다음ㅁ HTTP 메세지가 클라이언트와 서버 사이에서 교환된다.그리고 마지막으로 tCP 연결이 끝난다. HTTP는 무상태성을 가지고 있는데, 그냥 과거 기록이 남지 않고, 그냥 요청에 따른 응답만 한다는 것이다.
