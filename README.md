# Computer Science Notes

혼자 공부하면서 정리한 **컴퓨터 과학(Computer Science)** 과목별 학습 노트 저장소입니다. Obsidian Vault로 관리하고 있으며, Markdown 노트와 강의 캡처 이미지를 함께 정리합니다.

효율적인 학습과 복습을 위해 과목별, 주제별로 내용을 나누어 두었습니다.

## 이런 분들에게 유용합니다

- CS 기본기를 다시 정리하고 싶은 개발자
- 컴퓨터공학 관련 강의를 듣는 학생
- 기술 인터뷰를 준비하는 취업 준비생

## 저장소 구조

- `.obsidian/`: Obsidian 설정
- `images/`: 노트에 첨부된 이미지
- `Templates/`: Obsidian 템플릿
- 과목별 폴더: 과목별 상세 목차는 각 폴더의 `README.md`에서 관리합니다.
