# Syntax

구문(Syntax)은 **문장이나 프로그램이 어떻게 구조적으로 구성되는지**를 의미한다.
프로그래밍 언어에서는 이를 “올바른 형태의 코드 구조”라고 볼 수 있다.

이러한 Syntax는 결국 **Grammar** 에 의해 정의된다.

## 문법의 구성 요소

문법은 다음 4가지 요소로 이루어진다.

- **Terminal : 더 이상 나눌 수 없는 실제 기호 (예: 키워드, 변수명, 숫자 등)

- **Non-terminal : 다른 기호로 치환될 수 있는 추상적인 기호 (예: expression, statement)
    
- **Production** : 비터미널을 어떤 형태로 바꿀 수 있는지 정의한 규칙
    
- **Start Symbol** : 문법 해석이 시작되는 출발점
